## Details

### 0.1.x — Registry foundation

Registry schema, role catalogs, fragment helpers, PHPUnit assertions, Flex recipe, consumer handbook

### 0.2.x — Registry evolution

Additional tier roles, registry schema refinements, handbook improvements, install profile updates, compatibility refinements

### 0.3.x — Registry tooling

Registry inspection helpers, registry export formats, registry debugging, catalog discovery APIs, developer diagnostics

### 0.4.x — Developer tooling

CLI utilities, registry validation commands, PHPUnit integration improvements, developer workflows, better error reporting

### 0.5.x — Registry validation

Schema validation, role validation, fragment validation, compatibility checks, registry linting

### 0.6.x — Component metadata

Extended registry metadata, component capabilities, dependency metadata, feature flags, registry introspection

### 0.7.x — Ecosystem integration

Cross-package registry synchronization, install profile improvements, package discovery, tooling integration, documentation generation

### 0.8.x — SDK maturity

Performance improvements, registry caching, developer experience refinements, extension hooks, internal cleanup

### 0.9.x — API stabilization

Public API review, deprecation cleanup, registry freeze, migration guide, documentation completion

### 1.0.x — Stable SDK

SemVer-stable registry schema, stable role catalogs, stable testing API, stable extension points, long-term compatibility
